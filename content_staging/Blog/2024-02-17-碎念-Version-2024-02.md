## Windows

本來在Windows10以後安裝Windows就不需要花什麼時間跟精神了,現在又因為往Linux靠齊(我也不知道是不是這個緣故)導致Windows的cli工具也變好用了,所以我現在安裝程式都是用`winget`直接安裝,倒也不是這樣安裝就比較炫,只是單純的依靠winget安裝的程式可以很輕鬆地靠`winget`來update

```bash
# 尋找XXX程式
winget search XXX

# 安裝XXX程式
winget install XXX

# 移除XXX程式
winget uninstall XXX

# 更新所有程式
winget update --all
```

## PhotonOS

這個其實沒什麼特別好設定的,就這次裝的5.0來說,基本上只要一開始IP設定是OK的,頂多就是

在`/etc/fstab`裡面加上要加掛的NFS folder,然後記得要安裝`docker-compose`(現在可以直接用`yum install docker-compose`安裝就好了)以及`systemctl enable docker.service`還有`systemctl enable remote-fs.target`讓機器開機會自己跑docker還有加掛NFS folder就好了

### Docker

經過測試,目前我怎麼測試`nfs-volume`都還是失敗的,所以我現在還是老方法的在本機上乖乖的建立docker volume然後用script自動備份到nfs folder內進行備份比較簡單
