# 事情經過

其實事情的經過是很簡單的,就是我的PhotonOS重開過了之後發現討論區不能正常啟動,仔細看了一下是docker的volumes印射沒有成功,加上我不只是重開機

我還因為原來的設定沒有設定restart,導致重開後沒有自動跑起來的情況下手賤下了`docker-compose up -d`

然後就很docker就很帥的幫我recreate container....

然後就沒資料爆掉了.....

# Root cause

事後調查的結果非常的令我傻眼貼在下面

## 這是我原來的docker-compose.yml的節錄

```yaml
...    
    volumes:
      - /var/lib/docker/volumes/phpbb/mariadb:/bitnami/mariadb'
...
```

## 這是我改過後正常的的docker-compose.yml的節錄

```yaml
...    
    volumes:
      - /var/lib/docker/volumes/phpbb/mariadb:/bitnami/mariadb
...
```

看到了嗎？就差了一個`'`符號....
